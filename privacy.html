Privacy Policy

This Privacy Policy explains how DH Creative Lab handles information for its creative support tools and related services.

Operator DH Creative Lab
Last Updated April 27, 2026

1. Scope of This Policy

This Privacy Policy applies to DH Creative Lab tools and related pages published through GitHub Pages and other DH Creative Lab-managed locations.

Covered tools may include, but are not limited to:

  • Canva Font Finder
  • Etsy Canva Listing Helper
  • Pinterest SEO-related tools
  • Other DH Creative Lab tools

2. Purpose of the Tools

DH Creative Lab provides tools designed to support Canva font searching, Etsy and Pinterest content creation, Pinterest SEO improvement, content organization, and related creative workflows.

3. Information We May Access

Some tools may work without collecting personal information. However, when Pinterest API features are used, the tools may access information from your own Pinterest account, depending on the permissions you grant.

This information may include:

  • Pin information
  • Board information
  • Pin titles
  • Pin descriptions
  • Image URLs
  • Destination or link URLs

4. How We Use Information

Information accessed through Pinterest API or related tool functions is used only for the following purposes:

  • Improving Pinterest SEO
  • Managing and organizing creative content
  • Reviewing, editing, or improving Pin titles and descriptions
  • Supporting content planning and posting workflows

We do not use this information for unrelated advertising, profiling, or resale purposes.

5. Sharing of Information

DH Creative Lab does not sell, rent, trade, or share accessed information with third parties.

Information is used only within the tool workflow requested by the user, unless disclosure is required by law or necessary to protect the security and integrity of the service.

6. API Tokens and Authentication

API tokens, access tokens, refresh tokens, client secrets, and other authentication credentials should not be stored in public repositories.

DH Creative Lab does not intentionally publish API tokens or authentication credentials in public GitHub repositories. Users should also avoid entering or committing sensitive credentials into public code, public pages, or shared files.

7. Data Storage

DH Creative Lab aims to minimize data handling. Where possible, information is processed only for the requested tool function and is not permanently stored by the website.

If a specific tool later introduces storage functionality, the tool page or this Privacy Policy may be updated to explain what is stored and why.

8. User Control

If you connect a Pinterest account or authorize API access, you may revoke access through your Pinterest account settings or developer app settings where applicable.

9. Third-Party Services

Some tools may rely on third-party platforms or APIs, such as Pinterest, Canva, Etsy-related workflows, GitHub Pages, or browser-based services. Your use of those platforms may also be subject to their own terms and privacy policies.

10. Changes to This Policy

DH Creative Lab may update this Privacy Policy from time to time to reflect tool changes, API requirements, or legal and operational updates. The latest version will be posted on this page with an updated “Last Updated” date.
